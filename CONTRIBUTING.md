# Contributing to pacboost

Thank you for your interest in improving pacboost.

## Guidelines

1. Focus on performance and low-level speed.
2. Maintain a clean, serious, and emoji-free interface.
3. Adhere to Rust best practices and safety standards.
4. Ensure all source files contain the GPLv3 license header.

## Development

Build the project:
```bash
cargo build --release
```

Run tests:
```bash
cargo test
```

## Pull Requests

Submit pull requests against the master branch. Provide detailed descriptions of changes and ensure all checks pass.
